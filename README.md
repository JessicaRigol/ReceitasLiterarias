# Receitas Literárias

## Requisitos
1. [Docker](https://www.docker.com/) instalado.
2. [Visual Studio](https://visualstudio.microsoft.com/) instalado com o workload de desenvolvimento para ASP.NET Core.

## Como Executar
1. Clone o repositório para sua máquina.
2. Na raiz do projeto abra o CMD e execute o comando para iniciar o banco de dados:
      docker-compose up -d
3. Startar o projeto
